Interface System Week 2

Interface System Prototype

For this iteration of my interactive system I added pages for both the micro project as well as the sketches. I was thinking of having different layouts for the different pages, but I felt that having a consistent style was useful for navigation (preventing confusion for the user). I added a clear hierarchy to the page so that the information is conveyed easily from most important to least important. The most important information was the heading 1 title, which lives on the top of the page. Adding a simple border helps define the title from the rest of the page. Below that is the title of the sketch/project so that the name of the sketch is not lost in the information. I believe having the sketch centered near the top makes it easier for users to understand that the main content of the page is the sketch itself. Before, the body text lived above the sketch, so the first thing the user would see was the journal entries. This layout stays true for the most part in the phone view, with the exception of the information being centered on the screen. This helps phone users see the information without having to scroll side to side or have their fingers block the text. I also changed the landing page of the project as I felt the last iteration was a little messy. I changed the layout by condensing the information into the key sections of the sketchbook. When one section is clicked/tapped it opens a menu that shows all the options under that section. This makes the information seem cleaner and more intuitive for a user to interact with. This also helps phone users as there are less sections to accidentally tap on and get lost in. Along with that I also changed the layout of the desktop view to also have a centered design as well. This helps with the consistency of viewing the webpage on different devices. In the future I would like to add more interactivity with the different sections by possibly having hover effects. I also might change the information so that the most important sections in the landing page live at the top of the page.
