February 23rd

Concept Summary

I want to create a cookie-clicker like game where the user clicks a button on screen. When the button is pressed a currency counter goes up. This money from clicking can be used to purchase an upgrade that makes getting more currency easier.

Variables and Input

let points = 0; Points would be the main currency of the game and clicking the button adds one point. This changes behavior by enticing the user to get a higher score. Buying the upgrades would also make a multiplier go up which can be: let mult = 1

Base Build

let points = 0; let mult = 1; when function mousePressed is used, let points += mult; This should add the multiplier on each click so that clicking now gives the multiplier amount.

if upgrade is bought, spend 25 points and mult++. This should add one to the multiplier making clicking easier.

Structural Growth

Some ideas that I have to grow this project would be possibly adding an 'auto-clicker' upgrade that would automatically give points every second. This could run on a timer variable.

March 2nd

Current Build

I was able to get the multiplier working as well as the timer for the 'combo'. The main variable is still the points, but I believe the multiplier is what makes the button fun to click.

Pressure

There is now a timer that starts once the user hits a 3x multiplier. This forces the user to click within 5 seconds or the multiplier will go back down to 0.

Future Additions

I would still like to add an auto clicker function that adds a point every second.
